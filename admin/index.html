Suggested Modules

右側貼上劇本並分析後，這裡會出現建議模組。你可以逐一加入，或一次套用到故事軸。

Story Flow

左欄就是故事順序。由上往下是主線節奏，拖曳卡片可改順序。

Module Palette

點一下小圖示，就會在左側故事軸新增模組。

Current Story

這些資訊會和模組一起存進資料檔，之後可再接到 LINE 實作層。

Script Analyzer

把劇本貼進來後按分析。系統會抓出 PIC、選項、對話密度，並在左邊生出建議模組。

Module Inspector

先從左側選一個模組。

Story JSON

這裡是目前故事的實作資料，可提供給 webhook 或後續編排流程使用。
